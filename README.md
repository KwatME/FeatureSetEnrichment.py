For set enrichment analysis.

## Install

### FeatureSetEnrichment.py (this)

```sh
pip install git+https://github.com/KwatME/FeatureSetEnrichment.py
```

### julia

```sh
brew install julia
```

### julia packages

```sh
julia
```

```julia
using Pkg: add

add(url="https://github.com/KwatME/FeatureSetEnrichment.jl")

add("PyCall")

add("DataFrames")

add("Pandas")
```
